# Amazon UI Clone

This project is a frontend clone of the Amazon website, built using HTML and CSS.  
It focuses on replicating the layout and design of a real-world e-commerce platform.

## 🔗 Live Demo
https://maimunasabir.github.io/Amazon-Clone/

## 🚀 Features
- Amazon-inspired homepage layout  
- Product sections and image gallery  
- Navigation bar and footer design  
- Clean and structured UI  

## 🛠️ Technologies Used
- HTML  
- CSS  

## ⚠️ Note
- This is a frontend-only project (no backend functionality)  
- Currently optimized for desktop view (not fully responsive)  

## 📌 Purpose
This project was created for practice to improve frontend development skills and understand real-world website layouts.
